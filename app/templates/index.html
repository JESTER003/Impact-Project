This project is a demonstration of various web tools built using Flask and Tailwind CSS. Each tool showcases different functionalities and concepts:

  • URL Shortener: Convert long URLs into short, manageable links using hashing techniques.
  • To-Do App: Manage your tasks efficiently with a stack-based approach.
  • Realtime Chat-App: Engage in real-time communication using graph-based concepts.
